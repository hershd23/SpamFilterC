# SpamFilterC
This is an implementation of a basic Spam Filter in C language.

## How to run
Download/Clone the given repository and run the given code through the command prompt/terminal.

NOTE: If a windows user ensure that a suitable distribution of a C compiler (preferable gcc/MinGW) is installed on your machine.

## Ideas/Algorithms Used
The following is an implementation of algorithms like Bag of Words(BOW), K-means algorithm and K-Nearest neighbours algorithm.
The accuracy on the english.txt file is above 95%.
